# 入力操作コントローラー（サンプルUI）

キーボード入力・マウスクリック・待機を手順として登録し、順番に実行するためのシンプルなWeb UIです。

## できること

- キーボード入力 / マウスクリック / 待機 の手順追加
- キー欄で実際にキーを押してショートカットを選択
- 手順の一覧表示と削除
- 手順の順次実行
- ループ実行
- 実行中キャンセル

> 注意: ブラウザのみで動作するデモです。OS全体の入力制御はできませんが、ページ上では `KeyboardEvent` を発火し、矢印キーはキャレット移動/スクロールとして実際に反映されます。

## 使い方

1. `index.html` をブラウザで開く
2. 操作を追加して手順を作る
3. 「開始」で実行
4. 必要なら「ループ実行する」を ON
5. 途中停止は「キャンセル」

## ローカルサーバーで開く場合

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` にアクセスします。
